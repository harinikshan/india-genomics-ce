# Privacy

Genetic Health CE is designed to run locally.

## Defaults
- Analysis runs on your machine.
- No telemetry is enabled by default.
- Uploaded files in web mode are stored in local temporary job directories.

## Your responsibilities
- Protect local files and backups containing genomic data.
- Avoid exposing the web app to untrusted networks.
- Remove generated reports if they contain personal information.

## Medical disclaimer
This software provides informational analysis only and is not a clinical diagnosis.
Always consult qualified medical professionals before making healthcare decisions.
